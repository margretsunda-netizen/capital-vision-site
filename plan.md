# Aura Capital: Professional Investment Website Plan

## Project Overview
A premium, high-trust investment platform focused on asset management and portfolio growth.

## Feature Breakdown
- **Landing Page**: 
  - Sticky glassmorphism Navbar
  - High-impact Hero with generated image background
  - Real-time stats ticker
  - Core Investment Pillars (Real Estate, Tech, ESG)
  - Interactive ROI Calculator
  - Multi-tier Portfolio Grid
  - Security & Compliance section
  - Contact & Footer

## Design System
- **Colors**: Deep Charcoal (#1a1a1a), Gold Accent (#d4af37), White/Light Gray for text.
- **Typography**: Inter (Sans-serif) for body, Playfair Display (Serif) for headings.
- **Animations**: Framer Motion for scroll-reveals, staggered lists, and hover effects on cards.
